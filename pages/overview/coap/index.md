# COAP

Protocolo de comunicação leve e eficiente para dispositivos IoT, permitindo que eles interajam com aplicativos baseados na web em um ambiente com restrições de recursos.

- Protocologo de comunicação
- JSON
- Similar ao HTTP
- POST, GET
- Leve (menos recurso)
